Amazon Ürün Arama Hakkında

Bu uygulama, Amazon Türkiye üzerindeki ürünleri aramak ve karşılaştırmak için geliştirilmiş bir web aracıdır.

Özellikler

  • Gelişmiş ürün arama
  • Çoklu sayfa tarama
  • Detaylı ürün bilgileri
  • Gerçek zamanlı fiyat takibi

Nasıl Kullanılır?

  1. Ana sayfadaki arama kutusuna ürün adını girin
  2. Taramak istediğiniz sayfa sayısını seçin (1-10 arası)
  3. Ara butonuna tıklayın ve sonuçları bekleyin
  4. Ürün detaylarını inceleyin ve karşılaştırın

Teknik Detaylar

Backend Python Flask
Frontend HTML5, Bootstrap 5, JavaScript
Veri Kaynağı Amazon Türkiye
Güncelleme Sıklığı Gerçek Zamanlı
